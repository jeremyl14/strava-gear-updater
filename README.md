# Strava Gear Updater

## Overview
Checks for cycling activities with commute flag. Updates gear assiciated with activity to keep totals accurate

## Features
- Automatically update gear information on Strava

## Installation
To install Strava Gear Updater, follow these steps:

1. Clone the repository:
    ```sh
    git clone https://github.com/yourusername/strava-gear-updater.git
    ```
2. Navigate to the project directory:
    ```sh
    cd strava-gear-updater
    ```
3. Install the required dependencies:
    ```sh
    npm install
    ```

## Usage
To use Strava Gear Updater, follow these steps:

1. Configure your Strava API credentials in .env file.
2. Run the updater script:
    ```sh
    npm start
    ```
